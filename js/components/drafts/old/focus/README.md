# focus
## Steps
1. Get frontal alpha and beta (f*)
2. Collect a buffer with the bandpower values
3. Find the mean of the buffer
4. Convert these means to a boolean value

## Values
- Alpha Min: 0.7
- Alpha Max: 4
- Beta Min: null
- Beta Max: 0.7
